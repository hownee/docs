## Objective
Ensure that an admin user can configure the instance details and set up the first admin user.

## Prerequisites
- You must have a valid admin account. If you do not have credentials, please request them from your system administrator.
- At least one instance must already exist or be created.

## Steps
1. Log in to your instance setup page: [https://your-instance-name.papyrus.hownee.com/setup/*](https://your-instance-name.papyrus.hownee.com/setup/*)
	- _Screenshot needed: Setup page before entering details._
2. Enter the required instance details (e.g., name, description, settings) and the first admin user details. Please enter the details you used below:

	```
	[Insert instance name, description, and admin user details here]
	```

3. Save the configuration and wait for the system to process the changes.
	- _Video recommended: Record the configuration process and confirmation message._
4. Verify that the instance details and admin user are correctly set up.
	- _Screenshot needed: Confirmation page or dashboard after setup._

## Expected Results
- Instance details and admin user are configured as expected.
- If configuration fails, an appropriate error message is displayed (please capture a screenshot if this occurs).
