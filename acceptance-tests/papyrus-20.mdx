## Objective
Validate that modules are intuitive and easy to use for all users.

## Prerequisites
- Modules must be enabled and accessible in the instance.
- You must have access to an ATP instance or a personal instance.

## Steps
1. Log in to the ATP Instance or your own instance for testing:
   - [https://atptestinginstance-papyrus.papyrus.hownee.com](https://atptestinginstance-papyrus.papyrus.hownee.com)
   - [https://your-instance-name.papyrus.hownee.com](https://your-instance-name.papyrus.hownee.com)
   - _Screenshot needed: Dashboard or main page with modules visible._
2. Navigate through the different modules and interact with them (e.g., create, modify, delete items). Please describe your actions below:

   ```
   [Describe actions performed in modules]
   ```

3. Observe the feedback provided by the modules for each action.
   - _Video recommended: Record the interaction with modules and feedback received._
4. Assess whether the modules are easy to navigate and understand.
   - _Screenshot needed: Example of a module in use._

## Expected Results
- Users find modules easy to navigate and understand.
- If usability issues are found, document them and capture screenshots or video.
