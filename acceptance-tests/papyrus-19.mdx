## Objective
Ensure that an admin user can complete the instance setup, save the configuration, and receive confirmation.

## Prerequisites
- You must have a valid admin account. If you do not have credentials, please request them from your system administrator.
- At least one instance must already exist or be created.

## Steps
1. Log in to your instance setup page: [https://your-instance-name.papyrus.hownee.com/setup/*](https://your-instance-name.papyrus.hownee.com/setup/*)
	- _Screenshot needed: Final setup page before saving configuration._
2. Review all configuration details and ensure everything is correct. Please enter any final configuration notes below:

	```
	[Insert final configuration notes here]
	```

3. Click the **Finish** or **Save** button to complete the setup.
	- _Video recommended: Record the final save action and congratulatory message._
4. Wait for the system to process the changes and display a congratulatory or confirmation message.
	- _Screenshot needed: Confirmation or congratulatory message after setup._

## Expected Results
- All setup steps are completed and the user receives a congratulatory or confirmation message.
- If setup completion fails, an appropriate error message is displayed (please capture a screenshot if this occurs).

**Tip:** If you encounter any issues, please document the error message and steps taken, and attach screenshots or a video for troubleshooting.
