## Objective
Validate that the Papyrus vitrine website is accessible for users with disabilities and meets WCAG 2.1 AA standards.

## Prerequisites
- You must be a guest or a logged-in user.
- Access to a screen reader and ability to test keyboard-only navigation.

## Steps
1. Go to the Papyrus Vitrine Website: [https://papyrus.hownee.com/](https://papyrus.hownee.com/)
	- _Screenshot needed: Homepage with accessibility features visible (if any)._
2. Use a screen reader to navigate the website. Please specify the screen reader and OS used below:

	```
	[Insert screen reader name and OS here]
	```

3. Test keyboard-only navigation (Tab, Enter, Space, etc.) to ensure all interactive elements are accessible.
	- _Video recommended: Record a session using keyboard navigation and screen reader._
4. Note any accessibility issues or barriers encountered.
	- _Screenshot needed: Example of any accessibility issue found._

## Expected Results
- The site meets WCAG 2.1 AA standards and is accessible to users with disabilities.
- If accessibility issues are found, document them and capture screenshots or video.
