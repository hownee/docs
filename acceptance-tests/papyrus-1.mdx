## Objective
Ensure that an admin user can successfully log into the Papyrus vitrine app and access all main sections.

## Prerequisites
- You must have a valid admin account. If you do not have credentials, please request them from your system administrator.

## Steps
1. Open your web browser and go to the Papyrus vitrine website: [https://papyrus.hownee.com/home](https://papyrus.hownee.com/home)
	![Papyrus Homepage](/images/papyrus-vitrine-home.png)
2. Click the **Login** button. You will be redirected to the login page: [https://papyrus.hownee.com/auth/login](https://papyrus.hownee.com/auth/login)
	![Papyrus Login Page](/images/papyrus-vitrine-login.png)
3. Enter your **PAPYRUS_WEBSITE_APP_ACCESS** credentials (username and password). If you do not have these, please enter your credentials below:

4. Click the **Login** or **Submit** button.
5. Observe the system response. If login is successful, you should be redirected to the main dashboard or homepage of the app.
6. Once logged in, try navigating through different sections of the app (e.g., Dashboard, Profile, Settings) to confirm access.
  ![Papyrus dashboard after login](/images/papyrus-vitrine-dashboard.png)
## Expected Results
- The system accepts your credentials and logs you in.
- You are redirected to the main dashboard or homepage.
- You can access all sections available to your user role.
- If login fails, an appropriate error message is displayed (please capture a screenshot if this occurs).
